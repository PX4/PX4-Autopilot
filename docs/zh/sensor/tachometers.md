# 转速计（转数计数器）

转速表（又称[也被称为转数计数器传感器](https://en.wikipedia.org/wiki/Tachometer#In_automobiles,_trucks,_tractors_and_aircraft)）可用于测量诸如旋翼、发动机或车轮等车辆转动部件的旋转速率。

:::info
当前 PX4 仅记录 RPM 数据：它没有用于状态估计或控制。
:::

本节列出了 PX4 支持的转速计传感器（更详细的文档点击链接）。
