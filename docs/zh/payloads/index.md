# 有效载荷和摄像机

载荷是指载具为达到用户或任务目标而携带的设备。
PX4支持广泛的载具有效载荷，包括各类照相机、货物、仪器等等。

有效载荷连接到飞行控制器输出端口，并且可以在任务中自动触发，也能通过遥控控制器或操纵杆手动触发，还能从地面站（通过 MAVLink/MAVSDK 命令）触发。
