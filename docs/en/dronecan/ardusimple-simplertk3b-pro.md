# ArduSimple simpleRTK3B Pro

The [ArduSimple simpleRTK3B Pro](https://www.ardusimple.com/product/simplertk3b-x5/) is a high-performance triple-band RTK GNSS receiver based on the Septentrio Mosaic-X5 module. It provides millimeter-level positioning accuracy with fast convergence times and excellent robustness in challenging environments thanks to advanced multi-frequency tracking and anti-jamming/anti-spoofing capabilities.

## Key Features

* Down to 10 mm GPS accuracy
* Up to 100 RTK position updates per second
* Anti-jamming and anti-spoofing capabilities
* The onboard XBee socket can be used to expand functionality with plugin accessories (CANbus, MR/LR/XLR radios, Bluetooth, WiFi, Ethernet, 4G, Dataloggers, RS232, CANbus, L-Band)

> **Info:** This GPS module is made in Europe and is NDAA compliant.
