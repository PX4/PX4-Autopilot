# ArduSimple simpleRTK2B Budget

The [ArduSimple simpleRTK2B Budget](https://www.ardusimple.com/product/simplertk2b/) is a low-cost dual-band RTK GNSS receiver based on the **u-blox ZED-F9P**.

## Key Features

* Down to centimeter GPS accuracy
* Up to 10 RTK position updates per second
* The onboard XBee socket can be used to expand functionality with plugin accessories (CANbus, MR/LR/XLR radios, Bluetooth, WiFi, Ethernet, 4G, Dataloggers, RS232, CANbus, L-Band).

> **Info:** This GPS module is made in Europe.
