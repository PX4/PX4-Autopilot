# ArduSimple simpleRTK3B Compass

The [ArduSimple simpleRTK3B Compass](https://www.ardusimple.com/product/simplertk3b-compass/) is a high-precision dual-antenna RTK GNSS receiver based on the Unicore UM982 module.

## Key Features

* Down to 10 mm GPS accuracy
* Heading accuracy of 0.1° per meter of baseline distance
* Up to 50 RTK position updates per second
* The onboard XBee socket can be used to expand functionality with Plugin accessories (CANbus, MR/LR/XLR radios, Bluetooth, WiFi, Ethernet, 4G, Dataloggers, RS232, CANbus, L-Band).
* Basic jamming immunity

## Hardware Specifications

| Specification                   | Value                                                                                 |
| ------------------------------- | ------------------------------------------------------------------------------------- |
| GNSS Module                     | Unicore UM982                                                                         |
| Accuracy (RTK position)         | 1.0 cm                                                                                |
| Accuracy (standalone)           | 1.2 m                                                                                 |
| GNSS heading accuracy           | 0.14° (1 m baseline), 0.03° (5 m baseline)                                            |
| Galileo HAS                     | Not supported                                                                         |
| SBAS accuracy                   | ~0.6 m                                                                                |
| Bands                           | L1 / L2 / L5                                                                          |
| Update rate                     | Default 1 Hz; up to 50 Hz (standalone/SBAS/GNSS+attitude), up to 20 Hz (RTK+attitude) |
| Form factor                     | Arduino UNO-style carrier (approx. 69 × 53 mm)                                        |
| Interfaces                      | Dual SMA antenna inputs, USB, UART (JST 6-pin), XBee expansion slot                   |
| Mounting method                 | 3 mounting holes                                                                      |
| RF antenna connector            | SMA (dual antenna input)                                                              |
| Plastic case                    | Available as an accessory                                                             |
| Operating / storage temperature | -40°C to +85°C                                                                        |
